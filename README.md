# Златоземье: Своя Земля

**Златоземье: Своя Земля** — Roblox-проект в жанре приключенческой RPG с развитием личного владения, экономикой и социальными активностями.

## Стадия проекта

Текущая стадия: **pre-production / MVP 0.1**.

На этом этапе репозиторий используется для подготовки базовой структуры, документации и минимального ядра кода без внешних зависимостей.

## Структура папок

```text
docs/                     Документация проекта и рабочие черновики дизайна.
src/                      Исходный код и структура объектов для Roblox Studio.
  ReplicatedStorage/      Общие модули, удалённые события и shared-логика.
    Modules/              Переиспользуемые Luau-модули.
    Remotes/              RemoteEvent/RemoteFunction-объекты и их описание.
    Shared/               Общие файлы для клиента и сервера.
  ServerScriptService/    Серверные скрипты и сервисы игры.
    Services/             Игровые сервисы: данные игрока, квесты, владения, валюта.
  StarterPlayer/          Клиентские скрипты игрока.
    StarterPlayerScripts/ Скрипты, запускаемые на клиенте.
      Controllers/        Клиентские контроллеры интерфейса и поведения.
  StarterGui/             Заготовки GUI для Roblox Studio.
  Workspace/              Заготовки объектов мира.
assets/                   Описание и будущие ассеты проекта.
```

## Работа с Roblox Studio

Код и структура из папки `src/` предназначены для переноса или синхронизации с Roblox Studio. На старте файлы можно переносить вручную в соответствующие сервисы Roblox, а позже подключить инструмент синхронизации, например Rojo, если команда примет такое решение.

Файлы `.server.lua` размещаются в серверных контейнерах, файлы `.client.lua` — в клиентских, а общие модули из `ReplicatedStorage/Modules` могут использоваться и сервером, и клиентом.
