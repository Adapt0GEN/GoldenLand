# 07. Next Steps

## Ближайший план разработки

1. **MVP 0.1.8 — UI ресурсов и уровня дома**
   - Показать игроку золото, дерево, камень и уровень дома.
   - Добавить короткие сообщения об ошибках и успешных действиях.

2. **MVP 0.1.9 — повторяемый сбор ресурсов после квеста**
   - Сделать деревья или ресурсные точки, которые можно собирать после завершения `first_steps`.
   - Продумать восстановление ресурсов без поломки текущего квеста.

3. **MVP 0.2.0 — второй уровень развития участка**
   - Добавить следующий этап владения после базового дома.
   - Подготовить визуальные элементы участка и новые требования к ресурсам.

4. **MVP 0.2.1 — простая мастерская**
   - Добавить первый функциональный объект на участке.
   - Связать мастерскую с ресурсами и прогрессом дома.

5. **MVP 0.2.2 — первый торговец**
   - Добавить NPC-торговца.
   - Подготовить простой обмен ресурсов или покупку базовых предметов.

6. **MVP 0.3.0 — дорога/портал в центральный город**
   - Добавить визуальный путь из стартовой зоны.
   - Подготовить переход к будущему социальному и торговому хабу.

## Главное направление

Ближайшие этапы должны развивать уже рабочий вертикальный срез, а не заменять его: сохранить цикл староста → квест → ресурсы → участок → дом → улучшение, постепенно добавляя повторяемые действия и новые причины возвращаться на личную землю.
