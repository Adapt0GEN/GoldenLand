## Текущая модель ForestZone

Логика открытия:

```text
ToolKitLevel >= 1
↓
игрок расчищает BlockedPathToForest
↓
ForestUnlocked = true
↓
открывается ForestZone
```

Поля профиля:

```lua
ForestUnlocked = false
ForestZoneState = "Locked"
ForestZoneClearedObjects = {}
```

Основные состояния:

```text
Locked — зона закрыта
Active — зона открыта и содержит объекты освоения
Empty — объекты освоения зоны очищены
```

Будущие состояния:

```text
Overgrown — зона открыта, но выглядит заросшей
Cleared — зона визуально освоена / преобразована
```

## Разделение типов объектов

```text
DecorOnly — только визуал
GlobalResourceNode — обычная ресурсная точка с респавном
ZoneClearObject — объект освоения зоны, очищается один раз
ZoneResourceNode — ресурс зоны, может респавниться, если так задумано
ZoneOneTimeReward — одноразовая находка или награда
```

Главное правило:

```text
Обычный ресурс:
добывается → временно исчезает → респавнится

Объект освоения зоны:
очищается → сохраняется как очищенный → не респавнится
```
