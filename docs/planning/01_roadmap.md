# MVP 0.4.x — Лагерь, город и производство еды

Цель: перейти от ручного выживания к развитию инфраструктуры.

```text
ферма
кухня
склад еды
простые блюда
сытость как бафф
городские производственные направления
первые элементы автоматизации
```

Боевые классы игрока не смешиваются с производственными профессиями.

```text
Класс = кем игрок является в бою.
Профессии = что город умеет делать автоматически.
```
