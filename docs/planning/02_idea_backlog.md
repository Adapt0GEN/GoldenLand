# 02. Idea Backlog

# IDEA-015 — Мрак как граница неизведанных областей

## Статус

Accepted / Future

## Категория

World Progression / Exploration Gate / Threat / Island Expansion

## Суть

Мрак — это неизведанные области острова, попадание в которые убивает игрока.

Мрак не является обычной стеной. Это опасная сила, которая ограничивает исследование и создаёт ощущение таинственного острова.

## Базовая механика

```text
игрок подходит к области Мрака
получает визуальное и текстовое предупреждение
если заходит слишком глубоко — погибает или переносится назад
Мрак можно рассеивать специальными сооружениями
часть Мрака может отступать по квестам
```

## Зачем это нужно

```text
контролировать темп исследования острова
не давать игроку слишком быстро открыть всю карту
создать ощущение опасности
дать смысл инженерным и магическим сооружениям
связать квесты с изменением мира
```

## Этап

MVP 0.6.x или позже.

# IDEA-016 — Инженерные сооружения для рассеивания Мрака

## Статус

Accepted / Future

## Категория

Engineering / Buildings / World Unlock

## Суть

Игрок может строить сложные инженерные сооружения, которые отодвигают Мрак.

## Примеры

```text
очистительная башня
маяк
энергетический пилон
защитный генератор
инженерный обелиск
```

## Стоимость

Должна быть высокой и требовать развитой экономики:

```text
дерево
камень
металл
золото
редкие ресурсы
детали
энергия
```

## Зависимости

```text
развитый лагерь
производственные цепочки
редкие ресурсы
новые зоны
возможно жители/специалисты
```

## Этап

MVP 0.7.x+.

# IDEA-017 — Магические сооружения для рассеивания Мрака

## Статус

Accepted / Future

## Категория

Magic / Buildings / World Unlock

## Суть

Часть Мрака можно рассеивать не инженерией, а магическими сооружениями.

## Примеры

```text
алтарь света
обелиск очищения
кристалл рассвета
ритуальный круг
священный огонь
```

## Почему это интересно

Позволяет развести две линии прогресса:

```text
инженерный путь
магический путь
```

И в будущем может поддержать разные стили развития города.

## Этап

MVP 0.7.x+.

# IDEA-018 — Квестовое отступление Мрака

## Статус

Accepted / Future

## Категория

Quest / World Change / Story Progression

## Суть

Некоторые зоны Мрака должны отступать не от построек, а от сюжетных действий игрока.

## Примеры

```text
спасти NPC
починить древний механизм
очистить лесную зону
победить источник порчи
доставить редкий ресурс
зажечь маяк
```

## Почему это важно

Игрок должен видеть, что квесты меняют остров.

## Этап

MVP 0.6.x+.
