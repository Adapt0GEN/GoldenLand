# IDEA-012 — Сытость как бафф-система

Статус: Accepted / Future.

```text
сыт — получает бонусы
не сыт — просто не получает бонусы
```

Бонусы:

```text
ускорение добычи
замедление роста усталости
ускорение восстановления усталости
увеличение длительности экспедиций
```

---

# IDEA-013 — Усталость как мягкий ограничитель экспедиций

Статус: Accepted for later.

```text
усталость растёт от активных действий
сытость снижает/замедляет усталость
дом восстанавливает усталость быстро
защищённая местность восстанавливает усталость медленнее
```

# IDEA-015 — Мрак как граница неизведанных областей

Статус: Accepted / Future.

Мрак — неизведанные области острова, попадание в которые убивает или возвращает игрока назад после предупреждения.

```text
Мрак можно рассеивать инженерными или магическими сооружениями.
Некоторые зоны Мрака отступают по квестам.
```

# IDEA-019 — Питомцы как спутники игрока

Статус: Accepted / Future.

Питомцы — маленькие спутники освоения острова.

Они могут:

```text
следовать за игроком
давать мягкие бонусы
реагировать на Мрак
помогать в исследовании
создавать эмоциональную связь
```

# IDEA-026 — Рюкзак и статы героя

Статус: Accepted / Future.

Рюкзак и статы должны появиться после стабилизации базовой экономики и зон.

Возможные статы:

```text
здоровье
выносливость
сила
магия
скорость
переносимый вес
уровень сытости
усталость
```

Рюкзак должен быть связан с:

```text
ресурсами
экспедициями
едой
редкими предметами
питомцами
```
