# 02. Idea Backlog

# IDEA-012 — Сытость как бафф-система

## Статус

Accepted / Future

## Категория

Food / Camp Economy / Buffs / Preparation

## Суть

Сытость не должна быть наказанием за отсутствие еды.

Сытость должна быть положительной системой баффов.

## Принцип

```text
сыт — получает бонусы
не сыт — просто не получает бонусы
```

## Возможные бонусы

```text
ускорение добычи
замедление роста усталости
ускорение восстановления усталости
увеличение длительности экспедиций
лёгкие бонусы к переноске ресурсов
```

## Связанные системы

```text
ферма
кухня
склад еды
еда как ресурс лагеря
жители лагеря
экспедиции
усталость
```

## Этап

MVP 0.4.x — Развитие лагеря и производство еды.

# IDEA-013 — Усталость как мягкий ограничитель экспедиций

## Статус

Accepted for later / Not Now for implementation

## Категория

Expedition / Rest / Soft Survival

## Суть

Усталость ограничивает долгие активные действия и дальние вылазки, но не должна раздражать игрока.

## Принцип

```text
усталость растёт от активных действий
сытость снижает/замедляет усталость
дом восстанавливает усталость быстро
защищённая местность восстанавливает усталость медленнее
```

## Что важно

```text
нет жёстких штрафов
нет блокировки игры
нет pay-to-play
```

## Возможные источники усталости

```text
добыча ресурсов
дальние экспедиции
опасные зоны
ночные события
бои
перенос тяжёлых ресурсов
```

## Возможное восстановление

```text
сон в доме
отдых в защищённой зоне
еда
улучшение дома
специальные постройки лагеря
```

## Этап

MVP 0.5.x — Угрозы, экспедиции и усталость.

# IDEA-014 — Связка сытости и усталости

## Статус

Accepted / Future

## Категория

Food / Fatigue / Expedition Prep

## Суть

Сытость должна быть одним из способов управления усталостью.

## Формула дизайна

```text
сытый персонаж медленнее устаёт
сытый персонаж быстрее восстанавливается
еда помогает готовиться к дальним вылазкам
город/лагерь производит еду для экспедиций
```

## Что не делать

```text
не делать голод как наказание
не продавать напрямую "право продолжать играть"
не превращать усталость в раздражающий таймер
```

## Этап

MVP 0.4.x–0.5.x.
