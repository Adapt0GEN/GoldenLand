# 03. Decisions Log

# DEC-014 — Мрак является глобальным ограничителем экспансии

## Решение

Мрак — это не просто декорация и не обычная невидимая стена.

Мрак становится глобальной системой ограничения исследования острова.

## Поведение

```text
неизведанная зона покрыта Мраком
попадание в Мрак смертельно опасно
Мрак можно рассеивать сложными сооружениями
часть Мрака может отступать по квестам
```

## Причина

Мрак помогает:

```text
контролировать темп исследования
создавать ощущение опасного острова
дать смысл развитию лагеря
дать смысл инженерным и магическим сооружениям
связать квесты с изменением мира
```

## Важно

Мрак не вводить слишком рано.

Сначала нужны:

```text
стабильные зоны
визуальное изменение зон
первые экспедиции
базовая экономика
производственные цепочки
```

# DEC-015 — Рассеивание Мрака должно быть дорогим и значимым

## Решение

Сооружения против Мрака должны быть сложными в производстве.

## Причина

Если Мрак легко убрать, он не будет ощущаться как важный барьер.

## Требования

```text
высокая стоимость
несколько типов ресурсов
возможная зависимость от редких материалов
возможная зависимость от жителей/специалистов
возможная связь с квестом
```

# DEC-016 — Мрак может отступать не только от построек, но и от действий игрока

## Решение

Некоторые зоны Мрака должны отступать по квестам.

## Причина

Это усиливает ощущение, что игрок меняет остров.

## Примеры

```text
после очистки зоны
после спасения NPC
после починки древнего механизма
после победы над источником порчи
```
