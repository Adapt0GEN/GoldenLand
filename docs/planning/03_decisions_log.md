# 03. Decisions Log

# DEC-011 — Сытость должна быть баффом, а не наказанием

## Решение

Сытость проектируется как положительная система бонусов.

## Не делаем

```text
жёсткий голод
смерть от голода
запрет играть без еды
сильные штрафы за отсутствие еды
```

## Делаем

```text
сытость даёт бонусы
еда важна для подготовки
сытость помогает снижать усталость
```

## Причина

Игрок должен хотеть заниматься едой, а не чувствовать, что его заставляют.

---

# DEC-012 — Усталость должна быть мягкой системой экспедиций

## Решение

Усталость нужна не как наказание, а как способ сделать дальние вылазки и отдых осмысленными.

## Принцип

```text
усталость растёт во время активных действий
сытость снижает/замедляет усталость
сон в доме восстанавливает усталость
защищённые зоны восстанавливают усталость медленнее
```

## Важно

```text
не делать pay-to-play
не блокировать игру
не вводить систему слишком рано
```

# DEC-013 — Донат не должен продавать право играть

## Решение

Донат на усталость, если появится, должен быть очень мягким.

## Можно рассмотреть позже

```text
косметика кровати
косметика кухни
визуальные эффекты еды
мягкие удобства
небольшие ускорения восстановления без слома баланса
```

## Нельзя

```text
заплати, чтобы продолжить играть
заплати, чтобы снять жёсткий таймер
заплати, чтобы получить огромное преимущество
```
