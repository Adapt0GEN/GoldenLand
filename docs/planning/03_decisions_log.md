# 03. Decisions Log

---

# DEC-001 — Ресурсы должны быть базовой четвёркой

```text
Дерево
Камень
Металл
Золото
```

---

# DEC-002 — Набор инструментов не одноразовый

Используем:

```lua
ToolKitLevel = 0
ToolKitLevel = 1
```

---

# DEC-004 — ForestZone должна быть зоной с состоянием

Модель MVP:

```text
Locked
Active
Empty
```

Будущая модель:

```text
Locked
Overgrown
Active
Empty
Cleared
```

---

# DEC-007 — Разделить обычные ресурсы и объекты освоения зоны

```text
GlobalResourceNode
ZoneClearObject
ZoneResourceNode
ZoneOneTimeReward
DecorOnly
```

---

# DEC-009 — Empty не должен выглядеть как нетронутый лес

Если ForestZone перешла в Empty, визуал должен измениться.

---

# DEC-011 — Сытость должна быть баффом, а не наказанием

```text
сытость даёт бонусы
еда важна для подготовки
сытость помогает снижать усталость
```

Не делаем:

```text
смерть от голода
запрет играть без еды
сильные штрафы
```

---

# DEC-012 — Усталость должна быть мягкой системой экспедиций

```text
усталость растёт во время активных действий
сытость снижает/замедляет усталость
сон в доме восстанавливает усталость
защищённые зоны восстанавливают усталость медленнее
```

---

# DEC-014 — Мрак является глобальным ограничителем экспансии

Мрак — не просто декорация и не обычная невидимая стена.

```text
неизведанная зона покрыта Мраком
попадание в Мрак смертельно опасно
Мрак можно рассеивать сложными сооружениями
часть Мрака может отступать по квестам
```

---

# DEC-017 — Питомцы должны быть спутниками освоения, а не обязательным pay-to-win

Питомцы дают мягкую пользу, эмоциональную связь и коллекционирование.

```text
обычные питомцы добываются в игре
донатные варианты — косметика/скины/эффекты
донатные питомцы не должны быть сильнее игровых
```

---

# DEC-018 — Боевые классы отделены от производственных профессий

```text
Класс = кем игрок является в бою.
Профессии = что город умеет делать автоматически.
```

В игре два основных боевых пути:

```text
Воин
Маг
```

Добывающие и производящие направления развиваются через город, здания, рабочих и автоматизацию.

---

# DEC-019 — Гильдейские завоевания являются поздним ресурсным sink

Гильдейская экспансия в нейтральные земли должна сжигать избыток ресурсов через:

```text
строительство
снабжение
осады
ремонт
удержание
улучшения
сезоны
```
