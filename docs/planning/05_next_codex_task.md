# 05. Next Codex Task

# MVP 0.4.0 - кузница и первая производственная цепочка

## Контекст

MVP 0.3.4 завершен и протестирован. В игре уже работает базовый экономический контур:

- стартовые ресурсы `Gold`, `Wood`, `Stone`, `Metal`;
- UI ресурсов;
- сохранение профиля игрока;
- личный участок с домом, складом и мастерской;
- `ToolKitLevel`;
- инструменты I и II;
- открытие ForestZone при `ToolKitLevel >= 1`;
- очищение ForestZone до `Empty` с сохранением состояния;
- открытие RockZone при `ToolKitLevel >= 2`;
- сохранение `RockZoneUnlocked`;
- добыча `Stone` из `RichStoneNode`;
- добыча `Metal` из `MetalVein`;
- защита RockZone ресурсов от дублей;
- обновление UI после добычи;
- cooldown/respawn RockZone ресурсов;
- Admin tool;
- Action Preview UI.

Следующий шаг должен быть маленьким и выполнимым. Не нужно сразу расширять проект в боевку, классы, инвентарь, сытость/усталость или крупную RPG-систему.

---

## Цель MVP 0.4.0

Добавить первую простую производственную цепочку через кузницу:

```text
Metal -> MetalIngot
```

Игрок должен получить понятный следующий смысл добычи металла в RockZone: не просто накопить `Metal`, а переработать его в первый производственный ресурс.

---

## Рекомендуемый объем задачи

### 1. Добавить ресурс `MetalIngot`

Добавить новый ресурс в профиль игрока и UI ресурсов:

```text
MetalIngot
```

На этом этапе достаточно одного нового переработанного ресурса. Не добавлять расширенную систему предметов или инвентарь.

### 2. Добавить кузницу на личный участок

Кузница должна быть простой постройкой/станцией рядом с домом, складом и мастерской.

Минимальное поведение:

- создается на личном участке игрока;
- имеет `ProximityPrompt`;
- доступна после открытия RockZone или после `ToolKitLevel >= 2`;
- показывает действие через Action Preview UI.

### 3. Добавить простую переработку металла

Минимальный рецепт:

```text
3 Metal -> 1 MetalIngot
```

Поведение:

- если у игрока достаточно `Metal`, кузница списывает `3 Metal` и выдает `1 MetalIngot`;
- UI ресурсов обновляется сразу после переработки;
- профиль игрока сохраняет новый ресурс;
- если ресурсов недостаточно, игрок получает короткое сообщение.

### 4. Не усложнять производственную систему

В рамках MVP 0.4.0 не добавлять:

- очередь производства;
- таймеры плавки;
- рецепты высокого уровня;
- отдельный инвентарь;
- экипировку;
- боевые классы;
- боевку;
- сытость/усталость;
- питомцев;
- гильдии.

## Что проверить

1. Запустить Play.
2. Дойти до состояния, где `RockZoneUnlocked == true`.
3. Добыть металл в RockZone через `MetalVein`.
4. Подойти к кузнице.
5. Проверить, что Action Preview UI показывает действие переработки.
6. Использовать кузницу при наличии минимум `3 Metal`.
7. Убедиться, что `Metal` уменьшился на 3.
8. Убедиться, что `MetalIngot` увеличился на 1.
9. Убедиться, что UI ресурсов обновился.
10. Перезапустить Play.
11. Проверить, что `MetalIngot` сохранился.
12. Проверить, что дом, склад, мастерская, ToolKitLevel, ForestZone и RockZone не сломались.
