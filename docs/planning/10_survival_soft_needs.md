# 10. Soft Survival Needs: Сытость и усталость

## Главный принцип

Сытость и усталость не должны быть раздражающим наказанием.

Они должны усиливать игровые циклы:

```text
город производит еду
еда готовит игрока к экспедициям
сытость даёт баффы
сытость снижает/замедляет усталость
дом и защищённые зоны дают отдых
```

## Сытость

Не делаем:

```text
смерть от голода
жёсткие штрафы
запрет играть без еды
```

Делаем:

```text
сыт — получает бонусы
не сыт — просто не получает бонусы
```

## Усталость

Усталость — мягкий ограничитель долгих экспедиций.

```text
добыча ресурсов
дальние экспедиции
опасные зоны
бои
ночные события
перенос тяжёлых ресурсов
```

Восстановление:

```text
сон в доме — быстро
отдых в защищённой зоне — медленнее
еда — снижает или замедляет усталость
улучшения дома — повышают комфорт восстановления
```
