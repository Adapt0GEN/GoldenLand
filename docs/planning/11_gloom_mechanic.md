# 11. Мрак

## Суть

Мрак — неизведанные области острова, попадание в которые смертельно опасно.

Мрак:

```text
скрывает неизведанные области
ограничивает скорость экспансии
создаёт угрозу
делает остров таинственным
даёт смысл инженерным и магическим сооружениям
```

## Базовое поведение

```text
игрок видит область Мрака
при приближении получает предупреждение
если входит слишком глубоко — погибает или переносится назад
Мрак можно рассеивать
часть Мрака может отступать по квестам
```

## Способы рассеивания

### Инженерный путь

```text
очистительная башня
маяк
энергетический пилон
защитный генератор
инженерный обелиск
```

### Магический путь

```text
алтарь света
обелиск очищения
кристалл рассвета
ритуальный круг
священный огонь
```

### Квестовый путь

```text
очистить зону
спасти NPC
победить источник порчи
починить древний механизм
зажечь маяк
```

## Связь с зонами

```text
ForestZone очищена
↓
появился проход к новой области
↓
но дальше стоит Мрак
↓
нужен квест или сооружение
↓
Мрак отступает
↓
открывается новая зона
```
