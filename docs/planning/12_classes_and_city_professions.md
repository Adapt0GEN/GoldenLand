# 12. Classes and City Professions

## Главный принцип

```text
Класс = кем игрок является в бою.
Профессии = что город умеет делать автоматически.
```

Игрок не должен лично быть “шахтёром 100 уровня”, если в будущем у него есть город, рабочие, здания и производственные цепочки.

---

## Боевые классы

Основные пути:

```text
Воин
Маг
```

### Воин

```text
Страж
Берсерк
Дуэлянт
Командир
Охотник на чудовищ
```

### Маг

```text
Стихийник
Призыватель
Целитель
Тёмный маг
Архимаг
```

---

## Городские профессии

Профессии — не классы персонажа, а направления развития города:

```text
Лесное дело
Горное дело
Металлургия
Кузнечное дело
Строительство
Алхимия
Фермерство
Инженерия
Торговля
Оборона
```

---

## Переход по этапам

```text
Ранний этап:
игрок = выживальщик

Средний этап:
игрок = герой + строитель лагеря

Поздний этап:
игрок = правитель города / командир / магистр

Эндгейм:
игрок = развивает город, управляет экспансией, защищает территорию
```

---

## Автоматизация

Сначала игрок делает всё руками:

```text
сам рубит дерево
сам добывает камень
сам собирает металл
сам строит первые здания
```

Потом открывает городские системы:

```text
лесопилка добывает дерево
каменоломня добывает камень
шахта добывает металл
плавильня делает слитки
кузница делает оружие
мастерская делает инструменты
ферма производит еду
казармы готовят защитников
```

---

## Разделение развития

```text
боевое развитие — через класс
экономическое развитие — через город
территориальное развитие — через зоны острова
технологическое развитие — через здания и исследования
```
