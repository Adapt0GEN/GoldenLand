# Planning README

Эта папка нужна для управления идеями, roadmap и ближайшими задачами Roblox/Rojo проекта GoldenLand.

## Как пользоваться

Новая идея сначала попадает в:

```text
02_idea_backlog.md
```

Если идея меняет архитектуру или важное правило проекта, решение фиксируется в:

```text
03_decisions_log.md
```

Если идея хорошая, но её рано делать, она попадает в:

```text
04_not_now.md
```

Если идея становится частью плана, она отражается в:

```text
01_roadmap.md
```

Текущая следующая задача для Codex хранится в:

```text
05_next_codex_task.md
```

После реализации и тестов обновляются:

```text
00_current_state.md
06_changelog.md
```

## Важное обновление v2

Добавлены отдельные правила по зонам и ресурсам:

```text
07_zone_architecture.md
08_resource_rules.md
```

Главное правило:

```text
Обычные респавнящиеся ресурсы и объекты освоения зоны — разные сущности.
```

Не нужно давать Codex весь backlog. В Codex отправляется только ближайшая задача из `05_next_codex_task.md`.
